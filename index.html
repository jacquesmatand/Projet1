Les bases de Vim

Le curseur se déplace avec les touches fléchées ou les touches hjkl. gauche : touche h bas : touche j haut : touche k droite :touche l.
Pour entrer dans Vim (à l’invite $) tapez vim [Fichier] . Pour quitter Vim sans enregistrer, tapez :q ! . Pour quitter Vim en enregistrant, tapez :wq , ou également :x , ou encore ZZ.

  • Effacements:

    • Pour effacer du curseur jusqu’à la fin d’un mot tapez dw.
    Pour effacer du curseur jusqu’à la fin d’une ligne tapez d$.
    Pour effacer toute une ligne tapez dd.

  • Annulations :

    • Pour annuler des actions précédentes, tapez u minuscule, comme undo. Pour annuler tous les changements sur une ligne tapez U majuscule. Pour refaire tapez -R comme redo. Insertion : Taper o ouvre une ligne sous le curseur et y place celui-ci en mode insertion. Taper un O majuscule ouvre une ligne au dessus de la ligne où se trouve le curseur. Tapez un a (append) pour insérer du texte après le caractère où se trouve le curseur. Taper un A majuscule ajoute du texte à la fin de la ligne. Pour remplacer le caractère sous le curseur, tapez r (replace) suivi du caractère qui remplacera l’original. Taper un R majuscule active le mode remplacement jusqu’à ce que la touche soit appuyée pour en sortir.

Les Commandes Externes

Les commandes d’effacement, comme dd (delete), en réalité coupent le texte. Pour coller le texte en mémoire après la position courante du curseur, tapez p. Si une ligne complète a été effacée, elle sera placée sous la ligne du curseur. P majuscule colle le texte avant (ou au dessus) la position courante du curseur.
Le changement vous permet de changer l’objet spécifié, du curseur jusqu’à la fin de l’objet. Par exemple, tapez cw (comme change word) pour changer du curseur jusqu’à la fin du mot, c$ pour changer jusqu’à la fin d’une ligne.
Le format d’utilisation pour le changement est : [nombre] c objet, ou encore c [nombre] objet. Pour copier du texte tapez yw, ou y$, ou yy (yank), avec le même format que précédemment indiqué. Position, recherche et remplacement Ctrl-G affiche votre position dans le fichier et l’état de celui-ci. G vous place à la fin du fichier. Un numéro de ligne suivi de G vous place à cette ligne. Taper / suivi d’un texte recherche ce texte vers l’avant. Taper ? suivi d’un texte recherche ce texte vers l’arrière. Après une recherche tapez n pour trouver l’occurrence suivante dans la même direction ou Maj-N pour rechercher dans la direction opposée. Taper%lorsquelecurseurestsurdesdélimiteursdebloc(,),[,],{ou}déplace celui-ci sur le caractère correspondant.

Autres Commandes

: ! exécute une commande externe. : !ls affiche le contenu du dossier courant.
:!rm efface.
:w enregistre le fichier Vim courant avec le nom Fichier.
:#,#w enregistre les lignes # à # dans Fichier.
:r lit et inclus Fichier dans le fichier courant à partir de la posi- tion du curseur.
